# Summary

Laser cut version of Bewegingslab's 3D print version.

worldmap-tesselated.dxf has shapes nested and arranged for efficient use of material -- I cut each quadrant from a different sheet.

worldmap-thickness.dxf has them arranged as traced -- use for reference or for other applications.

worldmap-single-vector.dxf is original tracing of the image in Bewegingslab's thing, in case anyone finds it useful.
